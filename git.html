Make a new repo

  1. Open GitBash
  2. Navigate to the directory you want as a local repo
  3. Type: git init
  4. Type this to make sure there has been a change (you should see red text): git status
  5. Type: git add .
  6. Type: git commit -m "Initialize repo”
  7. Go to GitHub
  8. Make a new repo there with the exact same name as your local directory.
  9. Copy the URL for the repo:

  10. Go back to GitBash and type: git remote add origin url
  11. Type: git push origin master
  12. Enter your GitHub creds.
  13. To back to GitHub and refresh the page to make sure all your files are in the repo.
  14. To make a gh-pages live site:
    • In gitbash, in your master, type: git checkout -b gh-pages
    • Type: git push origin gh-pages
    • Enter your GitHub creds.
    • Go to GitHub to make sure the gh-pages is there and complete.
    • You can view your site at username.github.io/reponame.

Update a repo

After you make changes on your local files you need to push them to GitHub

  1. Open GitBash
  2. Navigate to your local directory
  3. Type this to make sure there has been a change (you should see red text): git status
  4. Type: git add .
  5. Type: git commit -m "Initialize repo”
  6. Go to GitHub
  7. Type: git push origin master
  8. Enter your GitHub creds.
  9. To back to GitHub and refresh the page to make sure your changes made it to the repo.
